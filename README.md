# SignBridge – AI-Powered Sinhala Sign Language Assistive Technology Platform

## Project Overview
This is the research project website for "AI-Powered Sinhala Sign Language Assistive Technology Platform for Deaf and Mute Children", developed at SLIIT.

## File Structure
```
ssl-project/
├── index.html                    ← Main website (fully updated)
├── README.md
└── assets/
    ├── css/
    │   └── main.css              ← Original CSS (unchanged)
```

## Setup Instructions
1. Copy all files from your original project's `assets/vendor/` folder into `assets/vendor/`
2. Copy your original `assets/img/` files (bg.jpg, stats-bg.jpg, cta-bg.jpg, logo files, team photos)
3. Open `index.html` in a browser

## Customization Notes
- Replace `[Supervisor Name]`, `[Co-Supervisor Name]`, `[Student Name]`, `IT21XXXXXX` with real names/IDs in index.html
- Replace placeholder email `signbridge.sliit@gmail.com` with real project email
- Replace team photo placeholders with actual photos in `assets/img/team/`
- Replace download `href="#"` links with actual Google Drive or document links
- The system architecture diagrams are embedded as inline SVGs — no external images needed
